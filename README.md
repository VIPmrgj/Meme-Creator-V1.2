# Meme-Creator-V1.2
一个纯前端、零依赖的 Meme 短视频生成器。上传一张图片或一段视频，选择特效和音效，3秒生成可下载的 MP4 视频。技术栈：HTML5 Canvas ·网络音频API ·MediaRecorder API 核心能力： 14 种视觉特效（黑白/负片/鱼眼/波浪扭曲/赛博朋克/像素化等） 6 种合成音效（低音冲击/笑声/警报等） 一键导出 MP4，无需服务器，所有计算在浏览器本地完成 协议：MIT 许可 ·仅供娱乐学习使用
Meme Creator v1.2 改动总结

  一、架构优化

  音效延迟加载
  - 87 条 base64 从 JS 对象字面量 → 87 个 <script type="text/plain"> 标签
  - 浏览器不再解析 3.7MB 的 JavaScript，只当纯文本存储
  - 用户点击时按需解码，带 Blob URL 缓存
  - 删除了启动时后台预加载全部时长的循环
  - 新增 convert-sounds.js 一次性转换脚本

  特效系统重构
  - 扁平数组 + switch → 注册表 {k, i, l, cat, fn} + 哈希表分发
  - 新增特效只需加一行对象 + 一个 FR 方法
  - 从 20 种扩充到 45 种，分三类 tab：动态(15) / 滤镜(20) / 装饰(10)
  - _applyOne 从 O(n) switch 改为 O(1) 查表

  三、功能增强

  本地音效导入
  - "我的音效" 第 4 个 tab，初始为 0
  - 支持批量上传 MP3/WAV 等
  - 同名跳过、>10MB 跳过、格式校验
  - 导入后自动切换 tab 并刷新列表

  错误处理全覆盖
  - 文件上传：格式/大小/损坏 三类拦截
  - 视频生成：try/catch 包裹 + 90s 超时 + 空结果检测
  - 浏览器兼容：MediaRecorder / AudioContext / captureStream 预检
  - 音效加载：解码失败静默降级（生成无声视频）
  - 所有错误走 Toast，不崩溃，不卡按钮

  四、Bug 修复

  - _fh / _fv 翻转：改 1-t*2 线性为 cos(t*π) 余弦曲线 + setTransform 重置矩阵
  - _mr 镜面反射：源裁剪区改用原图实际尺寸而非画布尺寸
